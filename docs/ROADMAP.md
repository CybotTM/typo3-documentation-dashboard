# Roadmap

## Iteration 1: Static operations map

- Data model for repositories, people, communication, rules and relations.
- Static dashboard generated from JSON.
- Curated people and who-to-ask mapping.
- Initial relationship graph.

## Iteration 2: Live GitHub operations dashboard

- GitHub API collector.
- Repository list, open issue pressure, open PR count.
- Readiness probes for README, CONTRIBUTING, CODEOWNERS, SECURITY and workflows.
- Staleness hints based on pushed date.

## Iteration 3: Governance/readiness map

- Optional branch protection probes.
- Optional ruleset probes.
- Optional team permission mapping.
- CODEOWNERS parser.
- Lifecycle classification.
- Stale pull request buckets.
