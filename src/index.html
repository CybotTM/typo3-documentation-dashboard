TYPO3 Documentation Repository Operations Map

Orientation for maintainers, new team members and contributors: which repository owns what, where work is queued, which repositories lack ownership signals, and who to ask. Generated GitHub facts are kept separate from human-curated responsibility data.

What this dashboard is for

Every count below is collected live from the GitHub API. Ownership, expertise and communication data is human-curated and never inferred from commit history. Where a fact is not known it is shown as unknown or marked inferred rather than guessed. Archived repositories are hidden by default across every tab.

Data provenance

Curated human data lives under data/curated/. Generated GitHub data lives under data/generated/. Generated files are overwritten by the nightly collector; do not hand-edit them except as seed fallback.

This dashboard distinguishes generated GitHub facts from curated responsibility data. Unknown maintainership is shown as unknown, and inferred classifications are marked, rather than guessed.
